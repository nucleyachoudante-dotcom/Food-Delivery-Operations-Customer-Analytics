End-to-End Food Delivery Operations Analytics using SQL, Python and Power BI
# Food Delivery Operations Analytics

## Objective

Analyze food delivery operations and identify factors affecting revenue, delivery performance, and customer satisfaction.

## Tools Used

- SQL
- Python
- Pandas
- Matplotlib
- Power BI

## Dataset

15,000 food delivery transactions.

## Project Workflow

1. Data Cleaning
2. SQL Analysis
3. Python EDA
4. Business Insights
5. Dashboard Creation

## Key Findings
##Insight 1

Premium customers generated significantly higher revenue than regular customers.

Recommendation:

Increase premium membership campaigns.

- Peak order hours identified.
- Premium customers generated higher revenue.
- Delivery delays reduced customer ratings.
- Long-distance deliveries had higher delays.

##Insight 2

Orders peak between evening hours, requiring more delivery partners.

Recommendation:

Optimize rider allocation during peak demand.

##Insight 3
Higher traffic and bad weather increase delivery times.

Recommendation:

Introduce dynamic ETA models.

##Insight 4
Delivery delays negatively affect customer ratings.

Recommendation:

Reduce late deliveries to improve customer retention.

##Insight 5
Certain restaurant categories contribute most of the revenue.

Recommendation:

Focus marketing efforts on high-performing categories.
